# Chat Handler



### Payload

```json
{
  "message": {
    "message_id": 0,
    "text": "halo",
    "chat": {
      "id": "chat-user-id"
    },
    "from": {
      "id": "chat-user-id",
      "name": "name"
    }
  }
}
```


### Response

```json
{
  "code": 0,
  "response": {
    "text": [
      "Hi, selamat pagi ..."
    ]
  }
}
```
